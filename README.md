# DataAnalysis-BecasToeic2012-2013
Un análisis de datos en base a Resultados del éxamen de inglés TOEIC antes y después del otorgamiento de las Becas de Inglés Corfo 2012-2013

Las preguntas que busco resolver con mi procesamiento a través de PowerBI son:
1- ¿Existió una mejora en el rendimiento de los alumnos dentro del examen TOEIC gracias a las becas?
2- ¿Existe una correlación entre el tipo de beca y la mejora?

Mi fuente de datos fue obtenida a través de la plataforma datos.gob.cl, a través del siguiente enlace: https://datos.gob.cl/dataset/4963

El proceso de limpieza y transformación fue realizado a través de un Jupyter Notebook con el navegador Anaconda, lenguaje Python junto a las librerias PANDAS y NUMpy.
